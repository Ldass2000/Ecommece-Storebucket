# Ecommece-

Storebucket is an e-commerce website that allows users to browse and purchase products online
## Tech Stack

**Tech:**   Spring Boot
            Thymeleaf
            Spring Security
            Hibernate Validator
            Spring JPA
            MySQL 8
            Bootstrap 5
            Sweet Alert
            FontAwesome
            FSLightbox
            TinyMCE
            JavaScript

**Deployment:** Horuku
